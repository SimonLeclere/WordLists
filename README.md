# WordLists
WordLists est une liste de mots disponible en plusieurs languages.

# Utilisation
Vous pouvez librement utiliser ces mots pour vos projets.
Il y a au total 3690 questions (le fichier Francais contient 1081 mots et le fichier Anglais 2609).

# Provenance des mots

Les mots francais proviennent de la liste de mots connus d’un enfant de 4 ans, d’après l’académie d'Amiens.
Je ne saurais retrouver la provenance des mots Anglais.
